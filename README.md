# 📌 Overview

ProductPulse AI v2 is a next-generation **AI-driven Product Intelligence Platform** designed to automate product research and competitive analysis.

Instead of manually reading thousands of customer reviews, ProductPulse AI collects real-time user feedback from the Google Play Store and leverages Google's Gemini Large Language Model (LLM) to generate structured, business-ready insights.

The platform helps organizations make faster, smarter product decisions by transforming unstructured customer feedback into meaningful intelligence.

---

# 🎯 Business Problem

Modern applications accumulate millions of customer reviews across app marketplaces.

Manually analyzing this data is:

❌ Time Consuming

❌ Expensive

❌ Error Prone

❌ Difficult to Scale

❌ Nearly Impossible for Product Teams

ProductPulse AI eliminates this bottleneck by automatically discovering customer sentiment, recurring issues, feature requests, and competitive insights using Artificial Intelligence.

# 🚀 Getting Started

## 📥 Clone Repository

```bash
git clone https://github.com/Etashs/ProductPulse.AI.git
```

Move into the project directory

```bash
cd ProductPulse.AI
```

---

## 📦 Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 Configure Gemini API

Create a **.env** file inside the project root.

```env
GEMINI_API_KEY=YOUR_API_KEY
```

---

## ▶️ Run the Application

```bash
uvicorn app.main:app --reload
```

Open your browser

```
http://127.0.0.1:8000
```

Interactive API Documentation

```
http://127.0.0.1:8000/docs
```

# 📡 REST API

## Compare Products

```http
POST /compare
```

Example Request

```json
{
    "url1":"https://play.google.com/store/apps/details?id=com.whatsapp",
    "url2":"https://play.google.com/store/apps/details?id=org.telegram.messenger"
}
```
