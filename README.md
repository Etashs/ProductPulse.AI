\# 📌 Overview



ProductPulse AI v2 is an AI-powered competitive intelligence platform that compares two Google Play Store applications using real customer reviews.



Instead of manually reading thousands of reviews, ProductPulse AI automatically extracts valuable insights including:



\- Executive summaries

\- Customer pain points

\- Feature requests

\- Rating comparison

\- Review comparison

\- AI-based winner prediction



The system enables product managers, startup founders, business analysts, researchers, and developers to quickly understand how competing applications perform in the real world.

\# 🚀 Getting Started



\## Clone Repository



```bash

git clone https://github.com/Etashs/ProductPulse-AI.git

```



\---



\## Create Virtual Environment



```bash

python -m venv venv

```



Windows



```bash

venv\\Scripts\\activate

```



Linux / Mac



```bash

source venv/bin/activate

```



\---



\## Install Dependencies



```bash

pip install -r requirements.txt

```



\---



\## Run Application



```bash

uvicorn app.main:app --reload

```



Open



```

http://127.0.0.1:8000

```
